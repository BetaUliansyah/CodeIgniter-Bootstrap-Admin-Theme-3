Introduction

CodeIgniter Bootstrap Admin Theme 3 combines Bootstrap Admin Theme 3 by VinceG and CodeIgniter-Bootstrap by Steven Lu into one new project. Hopefully, it can help you to get into your development phase faster.

And I'm sorry for the long namespace. It's such kind of waste :P

Installation

Derived from CodeIgniter, read CodeIgniter Installation for how to install using this zipball instead. If CodeIgniter Tutorial confused you (like I did), you might want to read this easy introduction to CI for more information.

Flavors

  • The master branch contains the original and includes Twitter Bootstrap and Font Awesome.
  • The cdn branch contains the same libraries but is hosted by BootstrapCDN.
